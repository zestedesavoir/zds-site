Le context processor pour trouver la galerie automatique (auto_upload_gallery.py)¶

zds.gallery.auto_upload_gallery.get_auto_upload_gallery(request: django.http.request.HttpRequest)¶

This context processor adds auto_update_gallery to context. The gallery is the « default gallery » on forums and comments. On publishable content edition, it’s the content-specific gallery. :param request: the http request to use :return: a dictionary with auto_update_gallery key
